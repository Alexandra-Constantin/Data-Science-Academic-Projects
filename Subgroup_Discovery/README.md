# Subgroup Discovery

Application of subgroup discovery techniques to identify statistically interesting and interpretable patterns in data.

## Overview

This project explores rule-based subgroup discovery using quality measures, coverage constraints, significance testing, and pattern team selection.

## Methods

- Single nominal target subgroup discovery
- Single numeric target subgroup discovery
- Relative lift and Cortana quality measures
- Pattern team generation
- Swap-randomisation significance testing
- Double regression target analysis

## Tasks

- Identified subgroups associated with income categories
- Analysed age-related subgroup patterns
- Evaluated subgroup quality using multiple quality measures
- Compared filtered and unfiltered subgroup sets
- Assessed statistical significance through randomisation testing

## Skills

Python, data mining, subgroup discovery, pattern mining, statistical testing, exploratory data analysis
