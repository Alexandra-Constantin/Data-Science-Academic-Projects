# Data Visualization

Projects focused on data visualisation and dimensionality reduction using synthetic and benchmark datasets.

## Overview

This repository explores techniques for visualising high-dimensional data and understanding complex data structures through dimensionality reduction methods.

## Methods

- Principal Component Analysis (PCA)
- t-Distributed Stochastic Neighbor Embedding (t-SNE)
- Data standardisation and preprocessing
- 2D and 3D visualisation

## Projects

### PCA Visualisation
- Generated synthetic datasets and visualised their structure in 3D.
- Applied PCA to obtain low-dimensional representations.
- Compared original and projected data structures.

### Cluster Visualisation
- Analysed multivariate Gaussian clusters with different covariance structures.
- Investigated how PCA captures cluster separation.

### Swiss Roll Analysis
- Applied PCA and t-SNE to nonlinear manifold data.
- Compared linear and nonlinear dimensionality reduction techniques.
- Evaluated the ability of each method to recover underlying data structure.

## Skills

Python, NumPy, matplotlib, scikit-learn, PCA, t-SNE, dimensionality reduction, data visualisation
