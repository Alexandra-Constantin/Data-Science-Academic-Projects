# Recommender Systems

Implementation of recommendation algorithms using collaborative filtering and matrix factorisation techniques.

## Overview

This project explores multiple approaches for predicting user preferences from sparse rating data.

## Methods

- User-based collaborative filtering using cosine similarity
- Item-based collaborative filtering using cosine similarity
- Matrix factorisation using gradient descent with regularisation
- Similarity-based neighbourhood selection

## Tasks

- Computed similarity matrices between users and items
- Predicted missing ratings using collaborative filtering
- Implemented latent factor models through matrix factorisation
- Evaluated recommendation quality on benchmark rating data

## Skills

Python, NumPy, pandas, scikit-learn, recommender systems, collaborative filtering, matrix factorisation, predictive modelling
