# Data Science Academic Projects

Collection of projects completed during my MSc in Statistics and Data Science, demonstrating practical experience in Python programming. Based on real-world and benchmark datasets, these projects emphasise both algorithmic implementation and applied data analysis.

Topics covered:
* Python programming and object-oriented design
* Data wrangling and exploratory analysis
* Machine learning (classification, regression, recommender systems)
* Data visualisation
* Dimensionality reduction (PCA, t-SNE)
* Stream mining algorithms
* Subgroup discovery
* Scientific computing and algorithm implementation
